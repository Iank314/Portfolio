Featured Project

Distributed Task Queue

A horizontally scalable task queue built on PostgreSQL with an async aiohttp broker and a multi-worker pool. Uses SELECT ... FOR UPDATE SKIP LOCKED for atomic concurrent job claiming, proven safe under load via integration tests that run against real PostgreSQL.

Features priority scheduling, exponential-backoff retries, job cancellation, and a dead-letter queue for permanent failures. A dashboard UI and metrics endpoint surface queue health, and full CI via GitHub Actions guards correctness on every push.

  • aiohttp
  • PostgreSQL
  • Python
  • Docker Compose
  • GitHub Actions

Featured Project

Book Recommender

A full-stack recommendation system with a FastAPI backend that aggregates and reconciles results from Google Books and Open Library, deduplicating across APIs and scoring each match on a 0–100 hybrid formula (relevance + popularity signals).

Uses TF-IDF vectorization and cosine similarity over preprocessed descriptions and tag encodings to recommend books from a user's saved library. Single-page frontend in vanilla JS with pagination, relevance badges, and a personal library tab.
